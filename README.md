# ☄️ Comet Tracker

**Comet Tracker** is an Android application designed to retrieve accurate coordinates of known comets for visual observation or astrophotography sessions.

It is especially useful when working with smart telescopes such as Seestar or similar systems.

---

## 📌 Features

* 🔭 Retrieve comet coordinates in real time
* 🌌 Integration with NASA JPL Horizons system
* 📍 Select observatory location using IAU observatory codes
* 🛰️ Ideal for astrophotography and visual observation sessions
* ⚡ Lightweight and easy-to-use interface

## 🧠 How it works

Comet Tracker connects to the **NASA JPL Horizons** system to fetch precise ephemeris data for known comets.

Users can:

* Input a comet designation
* Select their observing location via IAU observatory code
* Obtain updated positional data for accurate pointing
